GIT

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

Git is easy to learn and has a tiny footprint with lightning fast performance. It is a version control system that helps you keep track of changes made to your files and collaborate with others on projects

HTML Elements

HTML elements or Hypertext Markup Langauge elements are the building block of WebPages. Everything you see on the internet are creating using html commands. Each HTML elements has a starting tag, content of the page, and a closing tag. Here are some of the most commonly used HTML elements

<!DOCTYPE> <html> <head> <title>Title of the Document</title> <head> <body> <h1 to h6> <p> <br and hr>
GIT Commands

Here are some common GIT Commands:

> git init: Initialize a new Git repository in the current directory > git clone: Create a copy of an existing git repository > git add : Adds the changes working directory to the staging area to be commited > git commit: Record the changes done to the repository > git push: Sends the local commit to the remote repository > git pull: Fetch the changes done to the remote repository and merge them to the current branch > git log: Displays the history of commits in the repository

Flexbox| CSS-tricks

Flexbox is a CSS layout model that allows people to design complex and responsive layouts with ease. It provides a more efficient way to align and dostrobure space among items in a container even when the size is unknown.

Flexbox Examples:

.parent{ display: flex; height: 300px; } .child { width: 100px; height 100px; margin: auto; } .flex-container{ display: flex; flex-flow: row wrap; justify-content: space-around; } .wrapper{ display: flex; flex-flow: row wrap; }
WSL

WSL is a compatibility layer for running Linux binary executables on Windows. The compatibility layer allows Linux binary executables to run natively on a Windows Operating System.

WSL Commands

Command > wsl --install > wsl > logout/exit > clear > wsl --shutdown
APACHE

Apache is a open-source web server software widely used on Linux Systems, making it a crucial component of many linux-based web hosting environements.

Getting started with apache

After installing wsl type wsl and then type: > sudo apt update > sudo apt upgrade > sudo apt-get install apache 2 > sudo service apache2 start
